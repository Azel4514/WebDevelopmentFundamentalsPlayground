CSS Statments

Four main componets of CSS statments:

  1. Declaration:The whole CSS statment in curly braces.
  2. SelectorThe name of the part you want to style.
  3. Property:Properties to edit(e.g., witdh, height, background, ect.)
  4. Vale:Numbers, colors, URLs - things to give value to the property.
